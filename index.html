Mengapa Menggunakan GitHub Pages?

Platform yang sangat cocok untuk meng-hosting website statis, portofolio, atau proyek dokumentasi secara gratis.

Sangat Cepat

Langsung melayani file statis HTML, CSS, dan JavaScript Anda tanpa perlu konfigurasi server yang rumit.

100% Gratis & Aman

Sudah dilengkapi dengan sertifikat SSL (HTTPS) secara default untuk menjaga keamanan pengunjung website Anda.

Template ini didesain menggunakan Tailwind CSS sehingga tampil rapi di layar komputer, tablet, maupun smartphone.
